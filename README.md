# 📊 DataClean

A simple and interactive data cleaning tool built with Streamlit that transforms raw datasets into clean, analysis-ready data.

---

## 🚀 Features

- Handle missing values (mean / median / mode)  
- Remove duplicate rows  
- Drop empty & constant columns  
- Fix data types automatically  
- Standardize text (lowercase, trim spaces)  
- Detect outliers (IQR method)  
- Generate data quality insights  
- Visualize missing data & outliers  
- Download cleaned dataset  

---

## 🛠 Tech Stack

- Python  
- Pandas  
- NumPy  
- Streamlit  
- Plotly
